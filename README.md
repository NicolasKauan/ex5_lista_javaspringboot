# 📚 Lista de Exercícios - Spring Boot

Este repositório contém a implementação de exercícios práticos utilizando **Java com Spring Boot**, com foco em:

* Estrutura em camadas (Model, Repository, Service, Controller)
* API RESTful
* CRUD completo com Spring Data JPA
* Mapeamento de entidades com JPA

Contendo:

* `model` → entidades JPA
* `repositories` → interfaces JPA
* `services` → regras de negócio
* `controllers` → endpoints REST

## 🚀 Tecnologias Utilizadas

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
